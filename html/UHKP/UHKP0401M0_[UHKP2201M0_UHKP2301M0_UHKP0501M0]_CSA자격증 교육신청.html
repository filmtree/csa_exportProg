CSA 교육신청

노후준비상담사(CSA) 교육신청 코너입니다

중복 교육 수강은 불가합니다. 아래 교육 수료이력을 조회하시면 신청가능한 교육과정이 나타납니다. 이후 희망하는 교육과정을 선택하시고 교육일정 등을 확인해주세요.
교육을 신청하시려면 교육 일정을 확인 후 아래의 ‘교육신청’ 버튼을 눌러 순서대로 신청해주세요.

  • 교육 신청은 신청기간 안에 반드시 신청해주세요.
  • 교육 신청내역은 [마이페이지]에서 확인해주세요.
  • 본인 핸드폰 번호는 교육과정 문자안내(교육방법, 사이트 주소 등)에 반드시 필요하므로 정확히 입력해주시기 바랍니다.
  • (신청순서) 성명, 주민번호를 입력 → 과거 교육이력 확인 → 신청 가능한 교육과정 확인 → 희망교육 과정명 선택 → 교육기간, 시간 등 확인 → 교육대상, 시간 등 확인 → 교육대상, 핸드폰 번호 입력(필수) → 신청하기 → 개인정보 수집.이용 동의서 → 확인 → 신청완료!

CSA 2급(노후준비 서비스 제공자) 이러닝 교육과목은 아래 자격증을 보유한 경우 과목면제신청이 가능합니다.

아래의 표를 참고하시고 해당 자격증을 취득 후 보유하고 계신다면 관련 이러닝 과목을 면제할 수 있으니 선택 후 신청하시기 바랍니다. (자격증이 5개인 5개 전부 등록 가능)

  • CSA 2급(노후준비서비스 제공자) 교육만 해당됩니다.
  • CSA 교육신청 후 면제 과목을 신청해주세요.
  • 반드시 자격증(자격 취득을 확인할 수 있는 서류)를 스캔해 첨부해주세요. (PDF, JPG 파일 가능)

CSA 2급 이러닝 면제과목 신청

CSA 2급 이러닝 과목 중 보유자격에 따라 면제과목 신청이 가능합니다. CSA 1급, 보수교육, 시험 면제는 아닙니다. 아래 ‘행추가’ 버튼을 눌러 면제과목을 신청하세요!
자격증이 3개인 경우 3번 ‘행추가’ 후에 면제 신청 누르시면 됩니다. 신청한 내역 수정은 다시 조회하여 수정하고 싶은 것을 선택 후 삭제처리 해주세요!
