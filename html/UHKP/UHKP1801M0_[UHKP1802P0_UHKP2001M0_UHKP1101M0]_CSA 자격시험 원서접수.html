CSA 자격시험 원서접수

CSA 자격시험 원서접수를 할 수 있는 코너입니다.

CSA 1급 및 2급 교육과정 수료자는 아래의 본인사항 및 교육이력을 확인 후 CSA 자격시험을 신청할 수 있습니다! 신청서 접수 후에는 취소, 수정 등이 어려우니 고사장, 시험일 등을 반드시 확인하고 신청해 주세요!

  • 자격시험 신청은 우편접수도 가능합니다.
  • 아래 본인확인을 위해 성명, 주민번호를 입력 후 조회바랍니다.
  • CSA 교육 수료 이력이 없는 경우 자격시험 접수가 불가합니다.
  • (순서) 홈페이지 로그인 → 아래 본인 확인 → 교육과정 이력 확인 → 자격 시험 선택 → 시험정보(시험일, 고사장 등) 확인 → 필수값 입력 → 신청 → 수험표로 응시번호 확인
