자동 슬라이더(접근성준수)



.swiper-container,
.swiper-wrapper,
.swiper-slide		 스크립트용
(클래스명 변경하거나 없으면 기능작동 안함)
.btn-prev,
.btn-next,
.btn-stop		 컨트롤 버튼
(클래스명 변경하면 기능작동 안함. 생략가능. 기능적으로 필수요소는 아니지만 접근성을 위해 필요함)
반드시 슬라이더 적용한 타겟(.main-banner)의 내부에 있어야 한다. 위치는 자유롭게.
.btn-stop 클릭할때마다 토글됨(.btn-stop.active <--> .btn-stop)
slidesPerView가 있는 경우 commonUI.slider('', {} , 3); 세번째 옵션으로 같은 값을 넘겨줘야 한다.(접근성용)
마크업 구조

반드시 .swiper-container 를 감싸고 있는 부모 요소(슬라이더 타겟)에 적용해야 한다.

commonUI.slider('.main-banner', { 옵션 }); 
마크업 구조	

<div class="main-banner">
	<div class="swiper-container banner-slider">
		<div class="swiper-wrapper">
			<div class="swiper-slide"></div>
			<div class="swiper-slide"></div>
			<div class="swiper-slide"></div>
			<div class="swiper-slide"></div>
		</div>
	</div>
	<div class="banner-slider-nav">
		<div class="slider-page"></div>
		<div class="slider-controller">
			<button type="button" class="btn-stop">팝업존 <span class="toggle">자동움직임 멈춤</span></button>
			<button type="button" class="btn-prev">팝업존 이전 배너</button>
			<button type="button" class="btn-next">팝업존 다음 배너</button>
		</div>
	</div>
</div>

<script>
	commonUI.slider('슬라이더 타겟', { 옵션 자유롭게 } [, perView]);

	commonUI.slider('.main-banner', 
	{
		loop: true,
		speed:700,
		autoplay: {
			delay: 3000,
			disableOnInteraction: false,
		},
		navigation: {
			prevEl: '.banner-slider-nav .btn-prev',
			nextEl: '.banner-slider-nav .btn-next'
		},
		pagination: {
			el: '.banner-slider-nav .slider-page',
			type: "fraction",
			clickable: true,
		},
		observeParents: true
	});
</script>
